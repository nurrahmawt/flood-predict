README - Cara Menggunakan Aplikasi
Deskripsi

Aplikasi ini digunakan untuk memprediksi potensi banjir di Pekon Parerejo berdasarkan data cuaca.

Cara Penggunaan
  1. Klik "Lihat Prediksi Default" untuk menampilkan prediksi otomatis.
  2. Gunakan "Prediksi Manual" bila ingin memasukkan data curah hujan dan kelembaban sendiri.
  3. Pilih "Cek di OpenWeatherMap" untuk melihat peta cuaca secara langsung.
  4. Hasil prediksi akan muncul di bagian bawah halaman.
Catatan
  • Data cuaca default diambil dari lokasi Pekon Parerejo.
  • Pastikan koneksi internet aktif untuk mengambil data cuaca terbaru.
